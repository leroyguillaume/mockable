# mockable

This crate provides usefull traits to make easier to mock your code using (`mockall`)[https://github.com/asomers/mockall] crate.

## Getting Started

Add this to your `Cargo.toml`:

```toml
[dependencies]
mockable = "0.1.0"

[dev-dependencies]
mockable = { version = "0.1.0", features = ["mock"] }
```
